# Генератор постов о товаре для МП с возможностью редактирования и прямой публикации в VK (Flask + OpenAI + VK)

Веб-приложение для генерации продающих постов на русском языке по ссылке на товар, с возможностью:
- редактировать пост перед публикацией,
- сохранять посты в избранное,
- публиковать в сообщество ВКонтакте сразу или отложенно.

## Возможности

- Генерация поста по ссылке на товар (OpenAI).
- Выбор тона поста: дружелюбный / с юмором / экспертный.
- Редактирование сгенерированного текста (иконка карандаша) и сохранение изменений.
- Сохранение постов в избранное (локально в SQLite).
- Публикация в VK:
  - `Опубликовать сейчас`
  - `Отложенная публикация` с выбором даты/времени.
- Очистка служебной markdown-разметки перед отправкой в VK (например `###`, `Заголовок:`).
- История публикаций в VK (статусы `published`, `scheduled`, `error`).

## Стек

- Python 3.10+
- Flask
- OpenAI Python SDK
- requests, beautifulsoup4
- SQLite (файл `app.db`)

## Установка и запуск

### 1) Клонируйте/скачайте проект и откройте папку

```powershell
cd "C:\Users\User\Documents\Zerocoder\Vibe-code_Intensive\Site_Function"
```

### 2) Создайте виртуальное окружение

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 3) Установите зависимости

```powershell
pip install -r requirements.txt
```

### 4) Создайте `.env`

В корне проекта создайте файл `.env` со значениями:

```env
OPENAI_API_KEY=ваш_openai_api_key
FLASK_SECRET_KEY=любая_длинная_случайная_строка

VK_GROUP_ID=123456789
VK_ACCESS_TOKEN=ваш_токен_сообщества_vk
VK_API_VERSION=5.131
```

### 5) Запустите приложение

```powershell
python app.py
```

Откройте в браузере: [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Как пользоваться

1. Вставьте ссылку на товар.
2. Выберите тон поста.
3. Нажмите `Сформировать пост`.
4. При необходимости нажмите `✏️`, отредактируйте текст и нажмите `Сохранить`.
5. Дальше можно:
   - `Скопировать`,
   - `В избранное`,
   - `Опубликовать сейчас`,
   - `Отложенная публикация` + выбрать дату/время.

Важно: при отложенной публикации время должно быть в будущем (минимум около 10 минут).

## Подключение VK (кратко)

Нужны 2 ключевых параметра:
- `VK_GROUP_ID` — числовой ID вашего сообщества.
- `VK_ACCESS_TOKEN` — токен сообщества с правами на публикацию на стене (`wall`).

Токен и ID добавляются в `.env`.  
Публикация идёт методом `wall.post` от имени сообщества (`from_group=1`).

## Структура проекта

- `app.py` — backend (Flask), генерация, VK API, БД.
- `templates/index.html` — интерфейс.
- `static/style.css` — стили.
- `app.db` — локальная SQLite база (создаётся автоматически).
- `.env` — локальные секреты и настройки.

## База данных

Автоматически создаются таблицы:
- `favorites` — избранные посты.
- `vk_posts` — история отправок в VK.

## Типовые ошибки

- `Не найден OPENAI_API_KEY`  
  Проверьте `.env` и перезапустите приложение.

- `Не настроен VK: добавьте VK_ACCESS_TOKEN и VK_GROUP_ID в .env`  
  Добавьте параметры VK в `.env`.

- `VK: ...`  
  Проверьте валидность токена, права доступа и настройки сообщества.

- `Для отложенного поста выберите время минимум через 10 минут`  
  Укажите более позднее время.

## Безопасность

- Никогда не коммитьте `.env` с реальными ключами.
- Если ключ уже утёк, перевыпустите его в кабинете провайдера (OpenAI / VK).
