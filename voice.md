# Настройки стиля поста

Этот файл подсказывает, как быстро менять стиль генератора на базе OpenAI.

## 1) Общий тон
- По умолчанию: дружелюбный, интересный и продающий.
- Где менять в коде: функция `get_voice_settings()` в `app.py`, ключ `tone`.

## 2) Длина поста
- Сейчас: средняя (700-1000 символов).
- Где менять в коде: `get_voice_settings()` -> ключ `length`.

## 3) Количество эмодзи
- По умолчанию: умеренно (2-4).
- Где менять в коде: `get_voice_settings()` -> ключ `emoji_count`.

## 4) Стиль заголовка
- Сейчас: цепляющий.
- Где менять в коде: `get_voice_settings()` -> ключ `headline_style`.

## 5) Призыв к действию
- Где менять в коде: `get_voice_settings()` -> ключ `cta_style`.

## 6) Хэштеги
- Где менять в коде: `get_voice_settings()` -> ключ `hashtags_style`.

## 7) Модель OpenAI
- Текущая модель: `gpt-4o-mini`.
- Где менять в коде: функция `generate_post_with_ai()` в `app.py`, параметр `model`.
